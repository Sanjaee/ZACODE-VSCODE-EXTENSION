# Zacode Template Fullstack

Extension VS Code untuk clone cepat template Flutter, Go, dan Next.js dari Zacode.

## Fitur

- **Clone Flutter Template Login** - Template Flutter dengan sistem login siap pakai
- **Clone Go Template Login** - Template Go dengan sistem login siap pakai
- **Clone Next Template Login** - Template Next.js dengan sistem login siap pakai
- **Clone Next Drizzle Template Login** - Template Next.js dengan Drizzle ORM dan sistem login siap pakai

## Cara Menggunakan

1. Buka Command Palette (`Ctrl+Shift+P` atau `Cmd+Shift+P`)
2. Ketik salah satu command berikut:
   - `Zacode Clone: Clone Flutter Template Login`
   - `Zacode Clone: Clone Go Template Login`
   - `Zacode Clone: Clone Next Template Login`
   - `Zacode Clone: Clone Next Drizzle Template Login`
3. Pilih folder tujuan untuk clone repository
4. Tunggu proses cloning selesai

## Pengembangan

### Prerequisites

- Node.js
- npm atau yarn
- VS Code

### Setup

```bash
npm install
```

### Build

```bash
npm run build
```

### Package Extension

```bash
npm run package
```

### Publish Extension

```bash
npm run publish
```
